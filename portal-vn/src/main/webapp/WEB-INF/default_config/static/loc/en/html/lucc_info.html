Title Land cover change matrices
Quantity 27 (one per ecoregion and country level for the period 195-2000, 2000-2005, 2005-2010)
Data owner VNFOREST
Data provider UN-REDD Programme (LOA FIPI-FAO No. FAOR 09/2015 and LOA FIPI-FAO No. FAOR 19/2016)
Data producer FIPI consigned under UN-REDD Programme (as instructed by MARD) based on VNFOREST owned data sets.
Data production date 2016 (as deliverable from the LOA FIPI-FAO No FAOR 19/2016, used for the Modified FRL submission to the UNFCCC)
Format and resolution tables based on the spreadsheet 10 to 12 of the Viet Nam Forest Reference (Emissions) Level calculations Excel file
Description of the data Table presenting the number of hectares that have changed from one forest category to another for each 5 year period between two NFIMAP maps.
Remarks These matrices are generated by overlaying two consecutive forest cover maps. For example the matrix for the period 1995-2000 is prepared by overlaying the maps of 1995 and 2000.
