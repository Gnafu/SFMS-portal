Data Owner 1995-2010: Forest Inventory and Planning Institute (FIPI) 2015: FORMIS (NFI&S Programme)
Data Provider UN-REDD Programme
Date 2015 (Updated version for FRL submission)
Format and resolution Raster 200 x 200 meters, based on FIPI and FORMIS shapefiles.
Description of the data The forest cover change maps were prepared for each combination of forest cover maps by overlaying the forest cover raster files.

The changes between forest types were reduced to four categories. This dataset was prepared jointly between FIPI and the UN-REDD Programme.
