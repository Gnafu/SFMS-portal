Title Map of Viet Nam forest cover change per activity for the periods 1995-2000, 2000-2005, 2005-2010.
Quantity 3 (one map per period)
Data owner VNFOREST
Data provider UN-REDD Programme (LOA FIPI-FAO No. FAOR 09/2015)
Data producer FIPI consigned under UN-REDD Programme (as instructed by MARD) based on VNFOREST owned data sets
Data production date 2015 (as deliverable from the LOA FIPI-FAO No FAOR 09/2015, used for the FRL submission to the UNFCCC)
Format and resolution Raster 200x200 meters based on overlaying the forest cover maps and associating the changes between forest types to activities.
Description of the data The forest cover change maps were prepared for each combination of forest cover maps by overlaying the forest cover raster files (as described above). The changes between forest types were reduced to seven categories:
  • no change of category in Evergreen broadleaf forest (EBLF)
  • degradation in EBLF
  • restoration in EBLF
  • no change in non EBLF forest types
  • deforestation
  • reforestation
  • non-forest remaining non-forest
